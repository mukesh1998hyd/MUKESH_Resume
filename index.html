Projects

  • 1. Deploy a Multi-Tier Website on AWS

    Project Description:
    This project involves setting up a multi-tier architecture on AWS, which includes a web tier, application tier, and database tier. The goal is to ensure high availability and resilience of the website by leveraging various AWS services.

    Components:

    • Web Tier: Hosted on Amazon EC2 instances behind an Elastic Load Balancer (ELB).
    • Application Tier: Managed by an Amazon EC2 Auto Scaling Group.
    • Database Tier: Uses Amazon RDS with Multi-AZ deployment for high availability.

    Key AWS Services:

    • Amazon EC2: Compute instances for web and application tiers.
    • Amazon ELB: Load balancing across EC2 instances.
    • Amazon Auto Scaling: Automatically adjust the number of EC2 instances.
    • Amazon RDS: Managed relational database service.
    • Amazon Route 53: DNS service for domain management.
